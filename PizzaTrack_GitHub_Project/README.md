
# PizzaTrack 🍕

Sistema de gestión de pedidos de pizzería desarrollado en **Java** utilizando **pilas manuales con listas ligadas**.

## Objetivo

Simular un sistema de registro de pizzas que permita:

- Registrar pedidos
- Deshacer pedidos (Undo)
- Rehacer pedidos (Redo)
- Mostrar el pedido actual

## Estructura del Proyecto

src/
- Pizza.java
- Nodo.java
- PilaPizza.java
- GestionPedidos.java
- Main.java

screenshots/
- registro.png
- undo.png
- redo.png

docs/
- guion_video.txt

## Compilación

Abrir la terminal dentro de la carpeta src:

javac *.java

## Ejecución

java Main
